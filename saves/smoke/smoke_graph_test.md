# Marine Cooling Pump Fault Smoke Test

The seawater cooling pump on the auxiliary engine can fail because the suction filter is blocked.

When the suction filter is blocked, the common symptoms are:

1. Cooling water flow drops sharply.
2. Pump outlet pressure becomes unstable.
3. Auxiliary engine temperature rises.

The first maintenance action is to stop the pump safely, isolate the line, and clean the suction filter.

If the filter is clean but the flow is still low, inspect the impeller for wear and check whether air entered the suction side.
